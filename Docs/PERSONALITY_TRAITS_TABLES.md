# 武將個性資料表（正式版）

> 武將個性設計定稿。分類：**統御 · 戰鬥 · 智略 · 政治 · 特殊**。  
> 每個名稱**只出現一次**。合成個性以分號引用下級個性，實作時展開子效果。

### 分類規則

1. **合成個性**：效果欄引用其他個性名稱（如 `鬥將`、`屯田`）＝**包含**該下級個性的全部效果（實作時展開子個性）。
2. **跨類歸屬**：依下列順序判定，**命中即停**，不再往下：
   **統御 → 戰鬥 → 智略 → 政治 → 特殊**
   - 例：**宿將**（攻擊防禦＋單挑）→ 有戰鬥四圍 → **戰鬥**（非特殊）
   - 例：**一騎**（強運＋單挑）→ 無統御／戰鬥四圍 → **特殊**
   - 例：**據守**（使役＋再起）→ 使役含攻擊防禦 → **戰鬥**

3. **同類效果互斥**：展開合成後，若多個個性提供**同一下級引用**（如皆含 `強行`）或落在**同一數值槽位**（見下表），**只生效一個**（取色階較高；同色由實作定優先序，不疊加）。
4. **成對互斥**：部分個性**不可同時持有**，只能二選一。
5. **紫色名額**：每位武將在**每個大類**（統御／戰鬥／智略／政治／特殊）中，**最多只能擁有一個紫色個性**（該類的紫合成亦計入）。

色階：**紫**（合成）／**金**／**藍**／**紅**／**-**（未實裝）
