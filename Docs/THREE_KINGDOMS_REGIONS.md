# 三國時期各州都市與戰略名關完整清單（最終校正版）

## 1. 司隸（帝國核心、政治與經濟中樞）

### 都市

| 名稱 | 規模 | 行政 | 說明 |
|------|------|------|------|
| 洛陽 | 超大 | 河南尹 | 東漢、曹魏、西晉首都，天下的中心。 |
| 長安 | 超大 | 京兆尹 | 西漢舊都，曹魏防禦蜀漢的西北總指揮部。 |
| 河東 | 中 | 郡城 | 關羽家鄉，黃河下游的經濟與鹽業重鎮。 |
| 弘農 | 中 | 郡城 | 扼守關中平原與中原腹地的咽喉要道。 |
| 河內 | 中 | 郡城 | 司馬氏家族祖籍，緊鄰黃河的富庶郡城。 |
| 新安 | 小 | 縣城 | 洛陽西邊的軍事防禦屏障。 |

### 名關

| 名稱 | 說明 |
|------|------|
| 虎牢關／汜水關 | 洛陽的東面門戶。史實上為同一關口（汜水關為別名），演義中拆為兩關。三英戰呂布、關羽溫酒斬華雄、曹操大戰竇建德皆在此地。 |
| 潼關 | 曹操大戰馬超、韓遂之處。是關中平原的東面屏障，三國時期戰略地位逐漸超越老函谷關。 |
| 函谷關 | 長安與洛陽之間的鎖鑰。死卡在弘農郡的深谷中，是歷代關中與中原拉鋸的生死線。 |

## 2. 益州（蜀漢根據地、版圖遼闊的天府之國）

### 都市

| 名稱 | 規模 | 行政 | 說明 |
|------|------|------|------|
| 成都市 | 超大 | 蜀郡 | 蜀漢首都，西南第一大都市，天府之國的核心。 |
| 漢中 | 大 | 漢中郡 | 蜀漢北伐大本營，魏蜀爭奪的咽喉（南鄭）。 |
| 江州 | 大 | 巴郡 | 今重慶，長江水路物資與軍事集結的樞紐。 |
| 梓潼 | 中 | 郡城 | 劉備入蜀後新設的郡城，北上漢中的重要軍事中轉站。 |
| 涪城 | 中 | 縣城 | 今綿陽，綿竹關前方的戰略大縣，劉備與劉璋決裂時的第一個爭奪點。 |
| 建寧 | 中 | 郡城 | 諸葛亮南征平定的南中核心（味縣）。 |
| 江陽 | 中 | 郡城 | 長江水路要道上的重要物資郡城。 |
| 永安 | 小 | 縣城 | 即魚復縣（白帝城所在地），蜀漢東大門。 |
| 牂牁 | 小 | 郡城 | 南中邊疆郡城，偏遠但具備戰略牽制力。 |
| 越巂 | 小 | 郡城 | 西南夷交界處的軍事重鎮。 |

### 名關

| 名稱 | 說明 |
|------|------|
| 劍閣／劍門關 | 蜀中第一天險。諸葛亮在大小劍山之間修築。蜀漢滅亡前夕，姜維憑藉此關將鍾會十萬大軍死死擋在川蜀大門外。 |
| 綿竹關 | 成都的最後一道防線。位於涪城與成都之間，鄧艾偷渡陰平後，蜀漢諸葛瞻率軍在此與魏軍決死一戰，最終陣亡，綿竹失守宣告蜀漢政權壽終正寢。 |
| 陽平關 | 漢中的西大門。漢中防禦體系的核心，曹操征張魯、劉備爭奪漢中的死鬥之地。 |
| 葭萌關 | 漢中進入蜀中的第二道大門。位於梓潼北部，演義中張飛夜戰馬超、黃忠老將立功均在此關。 |
| 白水關 | 蜀中北防的重關，卡在白水匯合處，與葭萌關形成呼應。 |
| 陰平關 | 蜀中西北防線的邊郡關隘。魏滅蜀之戰中，鄧艾正是由此關出發，走無人涉足的「陰平小道」，翻山越嶺偷襲江油。 |
